# 教訓

* ABC、C問題がわからなくてもまずD問題を見てみるべし。自分のなかの難易度が世間の難易度と一致しないケースもあるので、Dのほうが簡単に解けることもある。
* 簡単なケースでは正常な動きをするが大きな入力で正しい答えがでないときは int で overflow している可能性を意識する。(10^9 + 7 で割った余り、みたいなときとか)
